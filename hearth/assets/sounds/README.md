# Sound Assets

This directory contains ambient soundscapes for the Campfire experience.

You'll need to add:
- `fire.mp3` - Crackling fire sound (loopable)
- `rain.mp3` - Gentle rain sound (loopable)

**Recommended sources:**
- Freesound.org (CC0 license)
- Zapsplat.com (free tier)
- YouTube Audio Library

Keep files under 1MB for performance.
